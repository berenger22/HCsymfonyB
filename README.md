# HCsymfonyB
Jeu éducatif en Symfony

Le projet s’inscrit dans le cadre de la formation du GRETA, le BTS SIO option SLAM.
Ce projet a était réailisé avec 2 collègues de formation.
Pour ma part, j'ai mis en place la partie Inscription, Connexion, Sécurité, et la page superAdmin avec un collègue.
Le but du projet "Harry Codeur" est de rendre les questionnaires de validation d’acquis amusants et ludiques.
